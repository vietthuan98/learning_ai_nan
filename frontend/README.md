# TinyLlama AI Chat - Frontend (Mesop)

A modern, responsive chat interface built with [Mesop](https://google.github.io/mesop/), featuring a persistent sidebar, multi-session management, and real-time streaming responses from a local TinyLlama backend.

## 🚀 Getting Started

### Prerequisites

- Python 3.10+
- The backend API server should be running on `http://localhost:8000`

### Installation

1. Navigate to the `frontend` directory:
   ```bash
   cd frontend
   ```

2. Create and activate a virtual environment:
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```

3. Install the dependencies:
   ```bash
   pip install -r requirements.txt
   ```

### Running the Application

Start the Mesop development server:
```bash
mesop main.py
```
Then open your browser at the URL provided in the terminal (usually `http://localhost:32123`).

## 📁 Project Structure

The project follows a modular component-based architecture for better maintainability:

- **`main.py`**: The main entry point and page layout assembly.
- **`state.py`**: Shared state management and global utilities.
- **`sidebar.py`**: Sidebar component with chat room list and creation.
- **`session_row.py`**: Individual chat session item with deletion logic.
- **`chat_main_area.py`**: Main chat interface with message streaming.
- **`api_client.py`**: Service class for interacting with the backend API.

## ✨ Key Features

- **Multi-Session Support**: Create, switch, and delete multiple chat sessions.
- **Real-Time Streaming**: Tokens appear as they are generated by the model.
- **History Synchronization**: Automatically loads chat history from the backend when switching sessions.
- **Smart Deletion**: Intelligently selects neighboring chats when a session is removed.
- **Responsive Design**: Clean sidebar layout with active session highlighting.
